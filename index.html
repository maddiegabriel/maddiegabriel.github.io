Flesch Index
Syllable Counter

This project can be downloaded from my github here:
github.com/maddiegabriel/syllableCounter

This program counts the number of words/syllables/sentences in a given text file. It also calculates the readability of the text using the "Flesch Index" calculation.

Please compile using the makefile:
>> type "make" in the terminal

Please run the program by typing:
>> ./fleschIndex filename

Where 'filename' is the name of a .txt filename containing a sentence/paragraph/etc.

To run with the sample text file, please type:
./fleschIndex whitman.txt

Reverse a Poem!

This project can be downloaded from my github here:
github.com/maddiegabriel/reversePoem

This program reverses the order of a given poem.

Please compile using the makefile:
>> type "make" in the terminal

Please run the program by typing:
>> ./reversePoem

When prompted, please type in a multi-line poem of your choice!
To terminate the program, please type a single period '.' on a newline.
It will output your poem in reverse order!

Sample Input:
----------------------------------------

Not to the swift, the race:
Not to the strong, the fight:
Not to the righteous, perfect grace:
Not to the wise, the light.
But often faltering feet
Come surest to the goal;
And they who walk in darkness meet
The sunrise of the soul.
Henry Van Dyke
.
----------------------------------------
